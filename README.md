# Titanic Survival Prediction using Machine Learning

## Overview
This project predicts passenger survival on the Titanic using machine learning classification models.

## Project Workflow
- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- Model Training
- Model Evaluation

## Tools Used
Python, Pandas, NumPy, Scikit-learn, Matplotlib

## Evaluation Metrics
Accuracy Score
Confusion Matrix
Cross Validation
